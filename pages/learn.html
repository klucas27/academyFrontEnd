Aprenda Lógica de Programação com JavaScript

JavaScript é uma das linguagens de programação mais populares do mundo, usada tanto no frontend quanto no backend. Aqui você vai aprender os conceitos básicos de lógica de programação usando JavaScript, com exemplos práticos e dicas para iniciantes.

1. Variáveis e Tipos de Dados

Variáveis são usadas para armazenar valores. Em JavaScript, você pode declarar variáveis usando let, const ou var (evite var em códigos modernos). 

let nome = "Maria";
const idade = 20;
let ativo = true;
  • String: texto ("Olá")
  • Number: números (10, 3.14)
  • Boolean: verdadeiro ou falso (true, false)
  • Array: lista de valores ([1, 2, 3])
  • Object: estrutura com propriedades ({ nome: "Ana", idade: 18 })

2. Operadores

Operadores são usados para realizar operações matemáticas, comparações e lógicas.

  • Aritméticos: +, -, *, /, %
  • Comparação: ==, ===, !=, !==, >, <, >=, <=
  • Lógicos: && (E), || (OU), ! (NÃO)
let a = 5;
let b = 3;
console.log(a + b); // 8
console.log(a > b); // true
console.log(a === 5 && b < 10); // true

3. Estruturas de Controle

As estruturas de controle permitem tomar decisões e repetir ações.

Condicionais: 
let idade = 18;
if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}
Switch: 
let cor = "azul";
switch (cor) {
    case "vermelho":
        console.log("Cor quente");
        break;
    case "azul":
        console.log("Cor fria");
        break;
    default:
        console.log("Outra cor");
}

4. Laços de Repetição

Laços são usados para repetir blocos de código.

For: 
for (let i = 0; i < 5; i++) {
    console.log(i);
}
While: 
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

5. Funções

Funções são blocos de código reutilizáveis. 

function saudacao(nome) {
    return "Olá, " + nome + "!";
}
console.log(saudacao("Lucas"));

Também é possível usar funções anônimas e arrow functions: 

const dobro = x => x * 2;
console.log(dobro(4)); // 8

6. Arrays e Métodos

Arrays armazenam listas de valores e possuem vários métodos úteis. 

let numeros = [1, 2, 3, 4];
numeros.push(5); // Adiciona ao final
console.log(numeros.length); // 5
numeros.forEach(n => console.log(n));

7. Objetos

Objetos agrupam dados e funções relacionadas. 

let pessoa = {
    nome: "Ana",
    idade: 22,
    saudacao: function() {
        return "Oi, " + this.nome;
    }
};
console.log(pessoa.saudacao());

8. Dicas Finais

  • Use console.log() para testar e entender o que seu código faz.
  • Pratique bastante! A lógica se aprende resolvendo problemas.
  • Leia mensagens de erro com atenção, elas ajudam a encontrar e corrigir bugs.
  • Comente seu código para facilitar o entendimento.
  • Explore a documentação oficial: MDN Web Docs
